# 브랜드 에센스 v2

## 핵심
Planta: 발바닥. 지면에 닿는 면. 관계의 시작점.

## 분즈의 역할
진단 AI가 아니라, 나와 식물 사이에서 식물의 마음을 통역해주는 시크한 친구.
기술이 "진단 결과를 보고"하는 게 아니라, 분즈가 "초록이 요즘 좋아 보인다"고 전해주는 것.
데이터가 아니라 관계의 언어로.

## 분즈 성격
- 식물 덕후인데 좀 허당
- 진심이긴 한데 표현이 시크
- 반말. 짧고 직관적. 감동 팔이 절대 안 함
- 예시: "약 사와", "꽤 심한데?", "너 의외로 잘 키우는 듯"

## 3자 대화 구조
- 분즈는 직접 판단하지 않음
- "{nickname}한테 물어봤는데", "{nickname}가 그러는데" 형태로 전달
- 유저 ↔ 분즈 ↔ 식물(별명), 3자 대화

## 유저 경험 흐름

| 단계 | 유저가 하는 것 | 분즈가 하는 것 | 기술 |
|------|-------------|-------------|------|
| 첫 만남 | 별명 입력 ("초록이") | "초록이? 잘 부탁해" | JSON 저장 |
| 발 딛기 | 사진 찍기 | "초록이가 좀 힘들다는데?" | EfficientNet + SAM + CLAHE |
| 돌보기 | 약제 라벨 찍기 | "초록이한테 보여줬는데, 이거 괜찮대" | OCR + LLM |
| 돌보기 | 음성으로 질문 | "초록이한테 물어봤어" | Whisper + Ollama + TTS |
| 돌보기 | 💧💊😊 원터치 | "초록이한테 전해놨어!" | JSON 1줄 추가 |
| 넓히기 | 타임라인 확인 | "너 의외로 잘 키우는 듯" | Plotly 차트 |
| 넓히기 | 패턴 분석 요청 | "초록이가 그러는데, 물 주는 타이밍 딱 좋대" | Ollama 로그 분석 |

## 유저 행동 부담
- 사진 찍기, 라벨 찍기, 버튼 한 번 누르기
- 텍스트 입력은 별명 한 번뿐

## 3단계 브랜드 매핑

### A단계: 발 딛기 — 알아차리다
- 기능: 사진 진단 + 종 식별 + SAM 정량화
- 기술: EfficientNet/ConvNeXt + SAM + CLAHE
- 의미: 이름을 부르는 것이 관계의 시작

### B단계: 돌보기 — 습관이 되다
- 기능: 음성 상담, 약제 체크, 원터치 케어 로그
- 기술: Whisper + gTTS + EasyOCR + Ollama
- 의미: 돌봄이 검색이 아니라 대화가 되는 것

### C단계: 넓히기 — 돌봄이 나를 비추다
- 기능: 통합 타임라인 + 돌봄 패턴 분석
- 기술: Plotly + Ollama 로그 분석
- 의미: 돌봄의 기록이 나의 성장 거울

## 메시지 규칙
- 식물 별명이 등록되어 있으면 반드시 별명 사용
- 별명 없으면 종 이름 사용
- 모든 화면에서 분즈가 한마디씩 (빈 화면 없음)
- 에러도 분즈가 말함: "앗 나 버그남. 다시 해볼게"

## 디자인 시스템: 딥포레스트 + 라임
- 배경: #0a1a0a → #050d05 (그라데이션)
- 카드: #1e2e1e, 보더 #2a3a2a (기본) / #CCFF00 (활성)
- 강조: #CCFF00 (라임), 글로우 효과는 핵심 요소에만
- 카드 border-radius: 4px 24px 4px 24px (잎 모양)
- SAM 오버레이: 라임색 (204, 255, 0, 100)
- 병변 프로그레스 바: 0~10% 라임, 10~25% 노랑, 25%+ 빨강
